# 编码 + 学习

“编码 & 学习”活动将让你随着 Node.js 核心贡献者们共同起步（或飞得更高）。无论你是你第一次还是第二、第三或是第四次为 Node.js 核心做出贡献，经验老道的贡献者们都能提供协助，他们也能现场即兴带领大家攻略 Node.js 核心源码。
